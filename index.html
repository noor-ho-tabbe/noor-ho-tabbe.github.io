最大回撤

在选定周期内任一历史时点往后推，于最低点时的收益率回撤幅度的最大值。最大回撤用来描述可能出现的最糟糕的情况。最大回撤是一个重要的风险指标，对于量化策略交易，该指标比波动率还重要。

  • $P_i$是某个时间点 i 的资产净值（或投资组合价值）。
  • $P_j$是在$i$之后的某个时间点$j$的资产净值($j > i$)
  • $max(P_i - P_j)$表示在所有可能的($j,i$)组合中，$P_i - P_j$的最大值

示例

时间 净值
第一天 100
第二天 110
第三天 90
第四天 80
第五天 95

计算过程

  1. 找出所有峰值和谷值组合：

    • 第1天 (100) → 第4天 (80):
      ( 100 - 80 = 20 )
    • 第2天 (110) → 第3天 (90):
      ( 110 - 90 = 20 )
    • 第2天 (110) → 第4天 (80):
      ( 110 - 80 = 30 ) （最大回撤）

MDD 越大，说明投资组合在历史上经历过的最大亏损越严重，风险越高。

Beta和Alpha

Beta（β）——系统性风险系数

相当于业绩评价基准收益的总体波动性

变量说明：

  • $r_p$: 投资组合收益率
  • $r_m$: 市场基准收益率
  • $Cov$: 协方差
  • $Var$: 方差

特征分析

β值范围 市场相关性 波动特征 典型资产
β > 1 高度相关 波动大于市场 科技股、成长股
β = 1 完全相关 与市场同步波动 指数ETF
0 < β < 1 正相关 波动小于市场 消费防御股
β ≤ 0 负相关 逆向市场波动 黄金、国债

Alpha（α）——超额收益能力

实际收益和按照Beta系数计算的期望收益之间的差额。代表策略多大程度上跑赢了预期的收益率。

参数说明：

  • $r_f$: 无风险利率（如10年期国债收益率）
  • $(r_m - r_f)$: 市场风险溢价

夏普比率和信息比率

夏普比率代表每多承担一份风险，可以获得几份回报，即单位风险所获得的超额回报，该比率越高，策略承担单位风险得到的超额回报越高，所以夏普比率越高越好。

变量说明：

  • $R_p$: 投资组合的平均收益率
  • $R_f$: 无风险利率（通常用国债收益率）
  • $\sigma_p$: 投资组合收益率的标准差（波动率）

经济含义

  • 分子：风险溢价（超额收益）
  • 分母：承担风险的代价（波动性）
  • 比值：单位风险获得的超额收益

评判标准

数值范围 风险收益特征 策略评价
> 1.5 优异的风险调整后收益 对冲基金顶级策略
0.5 ~ 1.5 合理补偿风险 公募基金主流水平
0 ~ 0.5 风险补偿不足 需优化资产配置
< 0 风险收益倒挂 立即止损

信息比率

含义与夏普比率类似，只不过其参照基准不是无风险收益率，而是策略的市场基准收益率。

变量说明：

  • $R_p$: 投资组合收益率
  • $R_b$: 基准组合收益率
  • $\sigma(R_p - R_b)$: 跟踪误差（Tracking Error）
